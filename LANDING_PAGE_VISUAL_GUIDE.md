## Color Palette Used

- **Primary Blue**: #2563EB (blue-600)
- **Primary Orange**: #EA580C (orange-600)
- **Success Green**: #16A34A (green-600)
- **Background Gray**: #F9FAFB (gray-50)
- **Text Gray**: #6B7280 (gray-600)
- **Dark Text**: #111827 (gray-900)

## Typography

- **Headings**: font-bold, varying from text-4xl to text-6xl
- **Body**: text-lg to text-xl
- **Small Text**: text-sm
- **Font Family**: System fonts (-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif)

## Spacing & Layout

- **Sections**: py-16 lg:py-24 (64px to 96px vertical padding)
- **Cards**: p-6 to p-8 (24px to 32px padding)
- **Gaps**: gap-4 to gap-12 (16px to 48px)
- **Container**: max-w-4xl to max-w-6xl (centered)

## Interactive Elements

- **Buttons**: 
  - Primary: gradient from blue-600 to orange-600
  - Outline: border-2 with hover effects
  - Size lg: px-8 py-3 text-lg
- **Cards**: hover:shadow-xl hover:-translate-y-1
- **Links**: hover:text-blue-700 with underline

## Responsive Breakpoints

- **Mobile**: Default (single column)
- **Tablet**: md: (768px) - 2 columns
- **Desktop**: lg: (1024px) - 3 columns or optimized layouts

## Key Visual Features

1. **Consistent Gradient Theme**: Blue to orange throughout
2. **Icon Usage**: Lucide React icons for visual interest
3. **White Cards**: on colored/gradient backgrounds for depth
4. **Popular Badge**: Orange badge on Starter pricing tier
5. **Trust Indicators**: Green checkmarks and colored dots
6. **Hover Effects**: Smooth transitions on all interactive elements
7. **Rounded Corners**: rounded-xl (12px) to rounded-3xl (24px)
8. **Shadows**: shadow-lg to shadow-2xl for depth
